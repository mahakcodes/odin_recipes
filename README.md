# odin_recipes
A multi-page recipe project demonstrating basic web development concepts, project organization, and version control, with room for future enhancements and additional features.
##Project Overview
This project is a multi-page recipe website built as part of The Odin Project Foundations course. The goal is to create a small, structured website to practice fundamental web development concepts.
##Skills Demonstrated
By completing this project, I have practiced and demonstrated:
-Structuring multi-page websites
-Organizing project files and folders
-Using links, lists, headings, and images
-Working with Git for version control
-Pushing and managing code on GitHub
##Future Scope / Reflection
This project lays the foundation for adding styling with CSS, interactivity with JavaScript, and further enhancing web development skills.
